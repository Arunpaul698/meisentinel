## What is Meisentis?

Meisentis automates the process of verifying whether a file or URL is safe to deploy or trust.

Most software gets installed, trusted, and forgotten. Nobody asks the hard question — *should this actually be here?*

**Meisentis does.**

Submit any file or URL. Meisentis scans it across 70+ antivirus engines, scores the risk from 0 to 100, generates an AI-written assessment summary, and produces a downloadable PDF report — all in minutes.

---

## Features

- **File & URL Scanning** — powered by VirusTotal's 70+ antivirus engines
- **AI Risk Scoring** — 0 to 100 with LOW / MEDIUM / HIGH classification
- **AI Assessment Summary** — plain-English report generated by Claude AI
- **PDF Report Export** — branded, downloadable report for every scan
- **4-Tier Pipeline** — hash registry → automated scan → AI analysis → analyst review
- **Auto-routing** — score-based triage with automatic resolution or escalation
- **Dark Industrial UI** — clean, professional web portal

---

## The 4-Tier Assessment Pipeline

| Tier | Name | Description | Time |
|------|------|-------------|------|
| T-1 | Hash Registry | SHA-256 checked against known-file registry | Instant |
| T-2 | Automated Scan | VirusTotal + 70 engines in parallel | 8–12 min |
| T-3 | AI Analysis | LLM confidence score + draft report | AI layer |
| T-4 | Analyst Review | Escalated for high-risk or ambiguous cases | Human |

---

## Risk Score Routing

| Score | Label | Action |
|-------|-------|--------|
| 85–100 | LOW RISK | Auto-resolved, ticket closed |
| 50–84 | MEDIUM RISK | Assigned to analyst with full draft report |
| 0–49 | HIGH RISK | Flagged immediately, specialist notified |

## Getting Started

### Prerequisites
- Python 3.11+
- VirusTotal API key (free at virustotal.com)
- Anthropic API key (pay per use at console.anthropic.com)

### Run Locally

```bash
# Set API keys
export VIRUSTOTAL_API_KEY="your_key_here"
export ANTHROPIC_API_KEY="your_key_here"

# Install dependencies
pip install -r requirements.txt

# Start backend
uvicorn main:app --reload --port 8000

# Open frontend
# Just open index.html in your browser
```

Backend runs at: `http://localhost:8000`
API docs at: `http://localhost:8000/docs`
